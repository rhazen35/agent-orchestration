# Decision Log

A permanent archive of completed tasks and their review notes. Exists to answer the question "why was this implemented this specific way?" months after the fact.

## Format

```
## [TASK-ID] Task Title
- Completed: YYYY-MM-DD
- Git: <hash>

### What was done
Brief description of the implementation.

### Review Notes
Any notes from the quality review — including rejections, iterations, and the reasoning behind the final approach.

### Why this approach
The rationale behind key implementation decisions.
```
